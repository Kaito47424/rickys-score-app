# GAS（Google Apps Script）

## ファイル構成

| ファイル | 用途 |
|----------|------|
| `rickys-api.gs` | スマホアプリ用Web API（doGet / doPost） |
| `main.gs` | スプレッドシート管理ツール（メニュー・エクスポート等） |

## GASへの追加手順（初回）

1. [Googleスプレッドシート](https://sheets.google.com) を開く
2. メニュー「拡張機能」→「Apps Script」をクリック
3. デフォルトの `コード.gs` を削除
4. 「＋ ファイル」→「スクリプト」で `rickys-api` を作成し `rickys-api.gs` の内容を貼り付け
5. 「＋ ファイル」→「スクリプト」で `main` を作成し `main.gs` の内容を貼り付け
6. 「保存」

## デプロイ手順（初回）

1. 右上「デプロイ」→「新しいデプロイ」
2. 種類: 「ウェブアプリ」
3. 設定:
   - **説明**: バージョンを記述（例: v1.0）
   - **次のユーザーとして実行**: 自分
   - **アクセスできるユーザー**: 全員
4. 「デプロイ」→ 表示されたURLを `app/.env.local` の `VITE_GAS_URL` に設定

## 再デプロイ手順（コード更新時）

1. GASエディタでコードを更新・保存
2. 右上「デプロイ」→「デプロイを管理」
3. 該当デプロイの鉛筆アイコン（編集）をクリック
4. バージョン: 「新しいバージョン」を選択
5. 「デプロイ」→ **URLは変わらない**

> ⚠️ 「新しいデプロイ」を押すと新しいURLが発行されVercelの環境変数更新が必要になる。
> 必ず既存デプロイの「編集」から再デプロイすること。
