# 要件定義書

## 更新ルール

- 新規要件はClaude.aiで設計・整理後にこのファイルに追記する
- ステータス: ✅ 完了 / 🔴 未対応 / 📋 設計済み / 💡 アイデア / 🔍 検討事項
- 優先度: 🔴 高 / 🟡 中 / 💡 低
- 更新時のブランチ名: `docs/update-requirements-YYYYMMDD`
